AniBridge Downloader Settings

General

Maximum number of simultaneous downloads (1-5). Lower values are recommended to avoid rate limiting.
Number of times to retry a failed download (0 = no retries). Uses exponential backoff between attempts and will try alternative providers on failure.
Automatically trigger a Jellyfin library scan when a download completes. This makes new episodes appear in your library immediately.
Allow non-admin users to access the AniBridge Downloader UI via the sidebar. Requires the File Transformation plugin (recommended) or falls back to direct index.html modification. A server restart is required after changing this setting.
Block new downloads while allowing the current queue to finish. Users will see a maintenance message and download buttons will be disabled.
Message displayed to users when maintenance mode is active.
HTTP proxy for all network requests and downloads (e.g. http://proxy:8080 or socks5://proxy:1080). Leave empty to connect directly without a proxy. A server restart is required after changing this setting.
Optional dedicated base directory for movie downloads across all sources. Leave empty to use the normal site/language download paths.
If the requested track (Sub or Dub) isn't available for an episode/provider, try the other one instead of failing outright.

AniWorld (aniworld.to) — English Sub

Enable searching and downloading anime from aniworld.to. English Sub only — aniworld.to does not offer an English dub track.
Base directory for downloads. Files will be organized as: Path/SeriesName/Season XX/SeriesName - SXXEXX.mkv
Default streaming provider for AniWorld. Vidmoly offers superior download speeds when available. VOE is a good fallback due to its wide availability.
If all retries with the preferred provider fail, the download will be retried with this provider.

s.to (SerienStream) — English Dub

Enable searching and downloading series from s.to. English Dub only — s.to does not offer an English sub track.
Override the s.to base URL. Use this if s.to does not resolve on your network. For example, set to https://serienstream.to as an alternative. Leave empty to use the default (https://s.to).
Base directory for downloads.
Default streaming provider for s.to.
If all retries with the preferred provider fail, the download will be retried with this provider.

Anikoto (anikoto.net) — English Sub & Dub

Enable searching and downloading anime from Anikoto. Offers both English Sub and English Dub. Built against a community-documented third-party API mirror (anikoto.net itself blocks automated access), so treat this integration as best-effort.
Leave empty to use the English Sub path above.

Anime Nexus (anime.nexus) — English Sub & Dub — Experimental

Disabled by default. anime.nexus has no publicly documented API and blocks automated research tools via robots.txt, so this integration is an unverified best-effort guess at the site's REST conventions. It may not work at all. Enable it only if you're willing to debug it — check the plugin logs for the actual HTTP responses if searches or episodes come back empty.
Leave empty to use the English Sub path above.
